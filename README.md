## Краткое описание решения

В рамках задания были сформулированы требования к новой функциональности личного кабинета продавца, позволяющей публиковать товар на маркетплейсе.

Модель включает:

1. верхнеуровневую постановку требования в формате **User Story**
2. описание сценария в формате **Use Case**
3. **Use Case диаграмму** взаимодействия продавца с системой
4. **диаграмму процесса публикации товара**
5. функциональные и нефункциональные требования к новой функциональности
