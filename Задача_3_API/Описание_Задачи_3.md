# Задача 3 – REST API интерфейс регистрации пользователя

## Краткое описание решения

В рамках задания был описан REST API интерфейс регистрации пользователя, вызываемый фронтендом при нажатии кнопки **Register**.

Решение включает:

- описание endpoint
- входные параметры
- успешный ответ
- ответы с ошибками
- примеры request / response
- OpenAPI спецификацию в формате YAML

---

## Используемые технологии и инструменты

- REST API
- HTTP / JSON
- OpenAPI 3.0
- Markdown
